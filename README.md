# DevOps End-to-End Project

## Overview
This project demonstrates a complete DevOps workflow starting from application development to production-grade deployment using CI/CD and Kubernetes.

## Tech Stack
- Python (Flask)
- Docker
- Git & GitHub
- Kubernetes
- GitHub Actions (CI)
- ArgoCD (CD)
- Prometheus & Grafana

## Application Endpoints
- `/` : Application home
- `/health` : Health check endpoint

## Goal
To build a production-ready DevOps pipeline following industry best practices.
